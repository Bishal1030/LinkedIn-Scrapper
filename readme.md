# LinkedIn Scrapper


## Setup

### 1. Clone the repo
```bash
git clone https://github.com/Bishal1030/LinkedIn-Scrapper
cd LinkedIn-Scrapper
```

### 2. Backend
```bash
cd backend
npm install
npm start
```

### 3. Frontend
```bash
cd frontend
npm install
npm run dev
```
